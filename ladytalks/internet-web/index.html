A história da Internet

Antes da Internet existir como a conhecemos, um longo caminho de evolução foi percorrido.

  • [1605] Francis Bacon desenvolveu uma codificação de letras do alfabeto em sequências de dígitos binários (0 e 1), conhecido como Cifra de Bacon, para esconder mensagens (criptografia). Mal sabia ele que 400 anos depois, estaria dando base para os princípios da tabela ASCII;

A história da Internet

  • [1666] O código criado por Bacon, depois foi aperfeiçoado por Gottfried Leibniz para o Sistema binário;
  • [1842] Ada Lovelace escreve o primeiro algoritmo para ser processado por uma máquina;
  • [1844] Primeira Transmissão de mensagens por telégrafo;
  • [1946] Criação do ENIAC, primeiro computador digital eletrônico;
  • [1957] A USSR envia para orbita o Sputnik para o espaço. Assim possibilitando as comunicações globais;

A história da Internet

  • [1958] Pesquisadores da Bell Labs inventam o modem (modulator-demodulator) que converte e desconverte sinais analógicos e digitias, assim possibilitando a comunicação entre computadores;
  • [1958] O Governo dos Estados Unidos cria a ARPA em resposta à USSR;
  • [1963] A tabela ASCII foi desenvolvida, permitindo que máquinas de diferentes fabricantes se comunicarem (primeiro padrão universal para computação);
  • [1966] Inicio do projeto ARPAnet. Que deu base para nossa Internet atual;

A história da Internet

  • [1972] Ray Tomlinson inventa o e-mail;
  • [1972] Criação da IANA -Internet Assigned Numbers Authority. Que cuida dos indereços IP e outras funções críticas;
  • [1973] Protocolo de transmissão TCP/IP começa ser desenvolvido;
  • [1974] Termo INTERNET é utilizado pela 1ª vez por Vint Cerf;
  • [1984 — 1989] Ben Segal convence CERN que TCP/IP chave para fazer a Internet funcional.
  • [1988] Criação do protocolo de comunicação via chat pela Internet: IRC ...

[1989] Proposta da Web (criação do protocolo HTTP) por Tim Berners-Lee

E que é a World Wide Web então?

  • É uma aplicação da Internet executada pelo protocolo HTTP (hypertext transfer protocol / protocolo de transferência de hipertexto);
  • É um sistema de documentos "hiperlinkados” entre si;
  • Os documentos hiperlinkados são no formato HTML;
  • Os arquivos HTML são armazenados em um servidor conectado na rede mundial de computadores (Internet);
  • Os documentos HTML são acessados via protocolo HTTP;

HTTP2 Lançado em Fevereiro de 2015

Após 16 anos sem grandes atualizações, o Internet Engineering Steering Group (IESG), órgão responsável por revisões técnicas nos padrões da internet, aprovou o HTTP/2 no último dia 18 de fevereiro.

HTML (Hypertext Markup Language / Linguagem de marcação de texto)

  • É uma linguagem aberta/livre de royalties criada especialmente para criar documentos hiperlinkados;
  • Hoje o HTML e vários outros padrões para Web como CSS, XML, CSV, WCAG, são mantidos e desenvolvidos pelo W3C - Consórcio da World Wide Web, também criado por Tim Berners-lee.
